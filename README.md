# 💬 Мессенджер

Полнофункциональное веб-приложение для обмена сообщениями в реальном времени.

## ✨ Возможности

- 🔐 **Аутентификация** - Регистрация и вход с JWT токенами
- 👤 **Профиль пользователя** - Редактирование имени, описания и загрузка аватара
- 🔍 **Поиск пользователей** - Быстрый поиск по имени или username
- 💬 **Чаты** - Создание приватных диалогов
- ⚡ **Сообщения в реальном времени** - WebSocket через Socket.io
- 📱 **Адаптивный дизайн** - Работает на мобильных и десктопе

## 🛠️ Технологии

### Backend
- **Node.js** - Сервер
- **Express.js** - REST API
- **MongoDB** - База данных
- **Socket.io** - Реальное время
- **JWT** - Аутентификация
- **bcrypt** - Хеширование паролей
- **multer** - Загрузка файлов

### Frontend
- **HTML5** - Разметка
- **Tailwind CSS** - Стили
- **Vanilla JavaScript** - Логика
- **Socket.io Client** - Реальное время

## 📋 Требования

- Node.js v20+
- MongoDB (локально или Docker)
- npm
